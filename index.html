                    • Filter nicknames with AutoMod. Don't want your members using certain words in their server nicknames? Set up nickname filters to enforce your server rules and stop members from chatting until they follow them.
                    • Members in small servers got mightier. For servers with fewer than 50 members, weve changed permissions so now everyone can add emoji, stickers, Soundboard sounds, and events. If things get too chaotic you can undo this change in Server Settings.
